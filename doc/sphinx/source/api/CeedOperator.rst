CeedOperator
**************************************

A `CeedOperator` defines the finite/spectral element operator associated to a :ref:`CeedQFunction`.
A `CeedOperator` connects objects of the type :ref:`CeedElemRestriction`, :ref:`CeedBasis`, and :ref:`CeedQFunction`.
