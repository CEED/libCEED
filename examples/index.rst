.. _Examples:

Examples
========================================

This section contains a mathematical description of all examples provided with libCEED
in the :file:`examples/` directory.
These examples are meant to demonstrate use of libCEED from standalone definition of operators to integration with external libraries such as `PETSc <https://mcs.anl.gov/petsc>`_, `MFEM <https://mfem.org>`_, and `Nek5000 <https://nek5000.mcs.anl.gov/>`_, as well as more substantial mini-apps.
