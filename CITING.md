# libCEED: How to Cite

```
  title =  {lib{CEED} development site},
  url =    {https://github.com/ceed/libceed},
  howpublished = {\url{https://github.com/ceed/libceed}},

```

The BiBTeX file for this reference can be found in the
`doc/bib/references.bib` file.
